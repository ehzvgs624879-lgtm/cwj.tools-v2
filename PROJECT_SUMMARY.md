# cwj.tools-v2 项目总结

## 项目概述
这是一个功能丰富的工具集，提供了文件操作、命令执行、会话保存等多种实用功能。

## 主要功能

### 1. 文件操作工具
- **文件读写**：支持读取、写入、追加文件内容
- **文件搜索**：在文件中搜索关键词及上下文
- **内容替换**：支持精准替换和批量替换文件内容
- **目录操作**：查看目录文件列表

### 2. 命令执行
- **终端命令执行**：可以运行各种系统命令
- **权限管理**：支持设置文件执行权限

### 3. 会话保存功能
- **状态保存**：保存当前工作目录、环境变量、文件列表等
- **时间戳记录**：自动生成时间戳标识
- **备注功能**：支持添加备注信息
- **历史管理**：保存和管理多个会话记录

## 文件结构

### 核心脚本
- `save_session.py` - 会话保存主程序
- `save` - 便捷命令脚本
- `install_save.sh` - 安装脚本

### 配置文件
- `config.json` - 配置信息
- `.bashrc_save` - bashrc别名配置

### 文档
- `README_save.md` - 会话保存功能使用说明
- `PROJECT_SUMMARY.md` - 项目总结文档

### 目录
- `sessions/` - 保存的会话文件目录
- `core/` - 核心功能模块

## 使用方法

### 基本使用
```bash
# 保存当前会话
save

# 保存会话并添加备注
save -"我的工作进度"

# 查看所有保存的会话
saves

# 查看特定会话
cat sessions/session_20260627_060754.json
```

### 高级功能
```bash
# 运行安装脚本
./install_save.sh

# 查看帮助
save -h
```
