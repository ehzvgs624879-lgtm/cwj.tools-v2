# CWJ TOOLS V2 — Design System\n\n> 版本：1.0 | 日期：2026-07-07\n> 状态：草案（基于现有代码分析）\n\n---\n\n## 一、设计哲学\n\n**\"Tools, not noise.\"**\n\nCWJ Tools 的设计基于以下原则：\n1. **工具即内容**——UI 不应喧宾夺主\n2. **暗色原生**——从暗色出发，Light 主题作为派生\n3. **微动效**——交互反馈轻柔而不张扬\n4. **移动优先**——但桌面端不应该是二等公民\n5. **一致性**——每个工具使用相同的交互范式\n\n---\n\n## 二、色彩系统\n\n### 2.1 当前提取色板\n\n| Token | 当前值 | 用途 |\n|---|---|---|\n| `--bg` | 工具图标颜色策略（未来改进）\n\n当前每个工具有独立渐变，未来改为按分类统一：\n\n```\n工具类别 → 统一渐变 → 保持视觉一致性\n\nutil（实用）:    --grad-util    → #2d3436 → #636e72\ncrypto（加密）:  --grad-crypto  → #0f3443 → #34e89e\ndev（开发）:     --grad-dev     → #4b6cb7 → #182848\nai（AI）:        --grad-ai      → #5f2c82 → #49a09d\nnet（网络）:     --grad-net     → #1e3c72 → #2a5298\n```\n\n---\n\n## 十一、无障碍 (a11y) 规范\n\n1. 所有交互元素需有 `:focus-visible` 样式\n2. 颜色对比度需达到 WCAG AA 标准\n   - 正文（`--text-primary` on `--bg-base`）：需 ≥ 4.5:1\n   - 大文字（≥18px 或 ≥14px bold）：需 ≥ 3:1\n3. 添加 `aria-label` 到图标按钮\n4. 面板打开时焦点应移入面板\n5. 按 Escape 键应关闭面板\n6. 颜色不能作为唯一的信息传达方式\n\n---\n\n## 十二、变更日志\n\n| 日期 | 版本 | 变更 |\n|---|---|---|\n| 2026-07-07 | 1.0 | 初始草案，基于 Phase 1 分析 |\n"}
